{{ define "info" }} Currency Converter.

Api V1

List Currencies(EURO) List currencies(Specific) Get rate(from -> to) Convert amount(from -> to)

Api V2

List Everything \o/ List currencies(Specific) Get rate(from -> to) Convert amount(from -> to)

I'm too lazy to sort some proper docs out, but the API is simple and only has 3 endpoints.

I built this so I didn't have to pay for another service.

Convert one currency to another:

GET

/conversion/{from}/{to} Ex: /conversion/gbp/usd

Links:

https://currencies.idt.dev/conversion/gbp/usd

Response:

{
  "base": "GBP",
  "currency": {
    "from": "GBP",
    "to": "USD",
    "rate": 1.415729
  }
}

Convert an amount to a different currency:

GET

/convert/{from}/{to}/{amount} Ex: /convert/gbp/usd/1000

Links:

https://currencies.idt.dev/convert/gbp/usd/1000

Response:

{
  "base": "GBP",
  "currency": {
    "from": "GBP",
    "to": "USD",
    "rate": 1.415729
  },
  "amount": 1415.729
}
